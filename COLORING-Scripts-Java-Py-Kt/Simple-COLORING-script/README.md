# Simple-COLORING-script-

Simple script to color word/s in Python.

`    PURPLE    = '\033[95m' # Purple`

`    GREEN     = '\033[92m' # Green`

`    YELLOW    = '\033[93m' # Yellow`

`    RED       = '\033[91m' # Red`

`    DEFAULT   = '\033[0m'  # Default`

`    BOLD      = '\033[1m'  # Bold`

`    UNDERLINE = '\033[4m'  # Underline`
